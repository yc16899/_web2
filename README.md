# 我的個人網站

這是一個使用 **Flask + CSS + JavaScript (Bootstrap 5)** 製作的網站，包含：

- 首頁
- 學習課程
- 工作經驗

## 🔗 Demo
你可以透過 **Render** 查看網站：

👉 [https://yc16899.github.io/_web1/](https://yc16899.github.io/_web1/)

## 📂 專案結構
_web2/  
│── app.py          # Flask 主程式  
│── README.md       # 說明文件 (給 GitHub)  

_web2/template  
│── home.html       # 首頁  
│── courses.html    # 學習課程  
│── jobs.html       # 工作經驗  

_web2/static  
│── image.png       # logo 照片  
│── style.css       # 共用樣式
